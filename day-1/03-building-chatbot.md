## 🤖 Hands-on Chatbot Construction
In this module, we built a modern AI assistant using **LangChain** and **Groq** to understand how LLMs can be utilized effectively.

### 📜 Technical Deep Dive
- **Provider**: [Groq](https://groq.ai) for high-performance inference.
- **Framework**: [LangChain](https://langchain.com) for prompt orchestration and chains.
- **LLM**: Using a heavy-duty model (`gpt-oss-120b`) for classroom assistant tasks.

### 📝 Key Features
- **Roleplay**: Setting the system prompt to act as an assistant to **Ruthran Raghavan, Chief AI Scientist**.
- **Prompt Templates**: Using `ChatPromptTemplate` for structured interaction.
- **Interactive Loop**: A basic REPL (Read-Eval-Print Loop) for real-time conversation.

Check out `chatbot.py` in this directory for the source code.
