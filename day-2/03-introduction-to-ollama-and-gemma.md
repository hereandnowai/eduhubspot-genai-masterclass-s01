## 💻 Running LLMs Locally
This module introduced the learners to the world of open-source models and their execution environments.

### 📥 Recommended Setup
- **VS Code**: The essential editor for AI development.
- **Ollama**: The most powerful way to run large language models locally on your machine.
- **Gemma 2 (1B/2B/27B)**: Google's lightweight, open-weights model derived from the Gemini technology.

### 🛠️ Key Topics
- **Ollama Commands**:
  - `ollama run gemma2:2b`: Running a specific model size for local development.
  - `ollama list`: Checking installed local models.
- **Model Variants**: Exploring Gemma-2 variants (27B, 9B, 2B) and embedding models for RAG applications.
- **Gemma 3 Preview**: Discussion on upcoming releases like Gemma-3 (270M) and its ultra-low latency capabilities.
